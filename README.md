# flipkart_ui

A full-featured **Flutter E-Commerce App** built using **GetX** for state management, **REST APIs** for product data, and **SharedPreferences** for persistent cart storage.

---

## Features

### Home Screen
- Product banners with auto slider
- Category-based product browsing

- Dynamic product loading from API

### Search
- Search products by **name**
- Real-time filtering
- **Highlighted search text**
- Smooth navigation to product details

### Product Details
- Product image mapped via product ID
- API-driven product details
- Price fetched dynamically from API
- Add to cart functionality

### Cart
- Add / remove products
- Increase / decrease quantity
- Cart data persists using **SharedPreferences**
- Dynamic total price calculation
